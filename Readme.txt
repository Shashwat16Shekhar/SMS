##Name
MEIJER - STORE MANAGEMENT SYSTEM

## Installation

python 3.9 or higher
pip

```bash
pip install flask
pip install cx_Oracle
```


## Usage
Import the DBProject, setup the database connection.
Edit the mainPage.py to enter your credentials for oracle (uname, pwd, dom)
Open Command prompt
Run the following bash commands (on windows)

```bash
set FLASK_APP=mainPage.py
python -m flask run
```

The command prompt will display the url for the app.
Open it on any browser.

Refer to the SMSFunctionalOverview-Screenshots file for overview of the application.
